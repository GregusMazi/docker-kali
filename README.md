**Mac/Linux**

```bash
docker run -t --user="$(id -u):$(id -g)" -v "$(pwd):/tmp" leplusorg/kali
```

Add the `--rm` option if you want to discard the container when you exit it.

**Windows**

In `cmd`:

```batch
docker run -t -v "%cd%:/tmp" leplusorg/kali
```

In PowerShell:

```pwsh
docker run --rm -t -v "${PWD}:/tmp" leplusorg/kali
```

Add the `--rm` option if you want to discard the container when you exit it.
